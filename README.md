# 暖心礼物 - 爱的告白网页

这是一个充满爱意的互动网页，当用户打开页面时会收到一个神秘礼物提示，点击确认后会弹出许多暖心的恋爱话语，铺满整个页面，并伴有温馨的背景音乐。

## 功能特点

- 精美的礼物提示弹窗，带有平滑动画效果
- 随机生成并分布的80条暖心恋爱话语，每条都有独特的动画效果
- 浪漫的背景爱心飘落动画
- 背景音乐播放功能（周杰伦的《告白气球》）
- 完全响应式设计，适配各种屏幕尺寸

## 技术栈

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion (动画效果)
- Vite (构建工具)
- pnpm (包管理器)

## 环境要求

- Node.js (推荐 v16 或更高版本)
- pnpm (包管理器)

## 安装与运行

### 1. 安装依赖

首先，确保您已经安装了pnpm。如果没有安装，可以使用以下命令安装：

```bash
npm install -g pnpm
```

然后在项目根目录下运行：

```bash
pnpm install
```

### 2. 运行开发服务器

```bash
pnpm dev
```

项目将在 http://localhost:3000 启动开发服务器。

### 3. 构建生产版本

```bash
pnpm build
```

构建后的文件将位于 `dist` 目录中。

## 音乐文件说明

项目中使用了周杰伦的《告白气球》作为背景音乐。由于版权原因，音乐文件可能需要您自行添加：

1. 在项目根目录创建 `public/runtime/music` 文件夹
2. 将音乐文件重命名为 `告白气球.mp3` 并放入该文件夹
3. 重新启动开发服务器或重新构建项目

## 项目结构

```
src/
├── components/      # 组件
│   ├── Empty.tsx    # 空状态组件
│   └── LoveMessage.tsx # 爱心消息组件
├── contexts/        # 上下文
│   └── authContext.ts # 认证上下文
├── hooks/           # 自定义钩子
│   └── useTheme.ts  # 主题切换钩子
├── lib/             # 工具库
│   ├── loveMessages.ts # 暖心话语数据
│   └── utils.ts     # 工具函数
├── pages/           # 页面
│   └── Home.tsx     # 首页
├── App.tsx          # 应用入口组件
├── index.css        # 全局样式
├── main.tsx         # 应用入口文件
└── vite-env.d.ts    # Vite环境类型定义
```

## 自定义修改

### 修改暖心话语

您可以在 `src/lib/loveMessages.ts` 文件中修改或添加更多的暖心话语。

### 修改背景音乐

背景音乐相关代码位于 `src/pages/Home.tsx` 文件中，您可以在其中修改音乐文件路径或添加其他音乐控制功能。

### 修改样式

项目使用Tailwind CSS进行样式管理，您可以直接在组件中修改Tailwind类来调整样式。

## 注意事项

- 音乐播放可能受到浏览器自动播放策略的限制，可能需要用户交互后才能播放
- 页面效果最佳体验请使用现代浏览器（Chrome、Firefox、Safari、Edge等）
- 如果您想在生产环境部署，请确保您有音乐文件的合法使用权
